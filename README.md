# 067 Inheritance

[![067 Inheritance](https://img.youtube.com/vi/V6rIqCEZV-A/0.jpg)](https://www.youtube.com/watch?v=V6rIqCEZV-A)

the index.js file
- should create a class named `Agable` that takes a `year` and has an `age` method that calculates the `age` (current year - objects year) _(use `new Date().getFullYear()` to get the current year)_
- should inherit from `Agable` in `Company` class and pass `yearEstablished` to `Agable` in the constructor
- should inherit from `Agable` in `Person` class and pass `birthYear` to `Agable` in the constructor
- should inherit from `Agable` in `Car` class and pass `year` to `Agable` in the constructor
